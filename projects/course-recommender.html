Overview

Developed a course recommendation system using UCSB’s course catalog and anonymized student enrollment data. The system suggests courses based on academic history, helping students explore relevant and challenging classes.

Tech Stack

  • Python (pandas, numpy)
  • R/RStudio
  • UCSB Academic Curriculumn v.3 API
